Contributing to Design Reference
Thank you for your interest in improving Design Reference! We welcome contributions that help make this knowledge base more valuable for the product design community.

How to Contribute

• Report Issues: If you find a typo, a broken link, or outdated information, please open an Issue in this repository.

• Suggest Improvements: Have a new UX method or tool to add? Open an Issue to discuss it.

• Pull Requests: If you want to make the changes yourself, feel free to fork the repository and submit a Pull Request.

Guidelines

• Please ensure all content is related to UX Methods, Cognitive Tools, or Leadership.

• Use a professional and inclusive tone.

• Ensure any external resources are properly cited.

By contributing, you agree that your work will be licensed under the same terms as the rest of the project.
