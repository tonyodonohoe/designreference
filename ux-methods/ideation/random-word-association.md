# Random Word Association

{% hint style="info" %}
AKA Random Attribute Association
{% endhint %}

Random word association is a creativity technique in UX design where designers generate ideas by connecting unrelated words. By linking diverse concepts, unexpected and innovative design solutions can emerge. Random word association stimulates out-of-the-box thinking, expands possibilities, and sparks new connections for unique user experiences.
