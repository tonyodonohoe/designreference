# SCAMPER

SCAMPER is a helpful ideation technique in UX design. It prompts designers to think creatively by questioning existing ideas and generating innovative solutions. Each letter in SCAMPER represents a different approach: Substitute, Combine, Adapt, Modify, Put to another use, Eliminate, and Reverse. SCAMPER encourages exploration and enhances design possibilities.
