# Eye Tracking

Eye tracking, in user testing, involves using specialised technology to measure and record the movement and gaze of a user's eyes as they interact with a website, application, or design. This technique provides insights into where users focus their attention, what they visually prioritise, and how they navigate visual elements. Eye tracking helps evaluate usability, visual hierarchy, and user engagement, aiding in optimising the user experience.
